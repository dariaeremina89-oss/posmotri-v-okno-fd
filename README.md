# Оно тебе надо

Учебный проект Ереминой Дарьи Андреевны в рамках курса Яндекс Практикума (Frontend-разработчик).

## Описание проекта

Одностраничный интерфейс с видеоконтентом и формой фильтрации.
Основная задача — реализовать стилизацию страницы строго по макету, не изменяя исходную HTML-разметку и JavaScript.

Проект направлен на отработку навыков верстки и работы с интерактивными состояниями интерфейса.

## Технологии

* HTML5 (предоставлен, без изменений)
* CSS3
* Flexbox
* БЭМ-методология

## Особенности реализации

* Верстка выполнена без изменения HTML и JS
* Видео адаптируется внутри контейнера с помощью `object-fit`
* Список карточек реализован с внутренним скроллом
* Интерактивные элементы имеют состояния hover, active и focus
* Для доступности используется `:focus-visible`
* Кликабельные элементы обозначены курсором (`cursor: pointer`)

## Структура проекта

* `index.html` — исходная разметка (не изменяется)
* `scripts/` — скрипты (не изменяются)
* `styles/` — стили проекта
* `images/` — изображения и иконки (не изменяются)
